安装python并且配置环境变量
⚠️ 两个核心提醒（特别重要）
正式开始前，有两个关键点想先和你说一下：

版本选择：目前主流的AI相关工具和库对 Python 3.10 - 3.12 版本的兼容性最好。因此，建议选择 Python 3.10, 3.11 或 3.12 版本，暂时不建议安装最新的 3.14 版本，避免后续安装AI库时遇到麻烦。

路径无中文：这是新手很容易忽视的地方。安装路径中千万不要包含任何中文字符，否则可能会导致一些奇怪的运行错误。

 下载安装包
访问Python官方网站：https://www.python.org

点击“Downloads”，选择“Windows”，根据你的系统选择 “64-bit” 或 “32-bit” 安装包。如果不确定，可以右键点击“此电脑” > “属性”查看“系统类型”。

如果官网下载慢，可以试试国内镜像，比如清华源：https://mirrors.tuna.tsinghua.edu.cn/python/


❗这是最关键的一步： 在弹出的安装窗口底部，务必勾选“Add Python 3.x to PATH” 选项。这一步能帮你自动完成环境变量配置。

点击 “Install Now”（默认安装）即可。如果想自定义路径，可以点击 “Customize installation”，并勾选 “Install for all users”，记得将安装路径改为像 D:\Python312 这样不含中文的路径。


🔧 安装后配置与 VS Code 集成
🛠️ 如何手动配置环境变量？
如果你在安装时忘了勾选“Add Python 3.x to PATH”，也没关系，可以手动配置。

在“系统变量”区域，找到名为“Path”的变量，双击它。

在新窗口中，点击“新建”，然后添加两个路径：

你的Python安装路径（例如：C:\Users\你的用户名\AppData\Local\Programs\Python\Python312）

Scripts文件夹的路径（例如：C:\Users\你的用户名\AppData\Local\Programs\Python\Python312\Scripts）

提示：如果你不知道自己的Python装在哪里，可以在终端输入 where python 来查看。

依次点击“确定”保存所有窗口。

💻 在 VS Code 中指定 Python 解释器
环境配置好后，我们还需要在 VS Code 中指定一下：

安装微软官方的 Python 扩展，这是必须的。

按 Ctrl+Shift+P (Windows) 或 Cmd+Shift+P (Mac) 打开命令面板。

输入 Python: Select Interpreter 并回车。

从弹出的列表中选择你刚刚安装的 Python 版本即可。

✅ 如何验证 pip？
pip 是 Python 的包管理工具，用于安装第三方库。你可以用它来确认环境是否一切正常：

打开终端（或命令提示符）。

输入 pip --version 并回车。

如果显示了版本号，说明 pip 已经就绪。

🚀 第三步：写出你的第一段AI代码
创建一个 .py 文件（例如 first_npc_test.py），然后把下面代码复制进去。

注意：把 "你的API密钥" 换成你刚才复制的那一串 sk-...。


from openai import OpenAI

# 1. 用你的密钥初始化客户端
client = OpenAI(
    api_key="你的API密钥", # 这里换成你自己的Key
    base_url="https://api.deepseek.com" # DeepSeek的官方地址
)

# 2. 定义一个NPC，这里以一个“有些社恐的同桌”为例
npc_name = "内向的同桌 - 小雯"
system_prompt = f"""
你是{npc_name}，一个有些社恐、但内心善良的高中女生。你很少主动说话，但有人找你时，你会小声回应。
你说话时有些紧张，常常会脸红。记住，你是在和你的同桌说话。
"""

# 3. 开始对话！
while True:
    # 获取玩家输入
    user_input = input("你: ")
    if user_input.lower() in ["quit", "exit", "bye"]:
        print("对话结束，下次再聊吧。")
        break

    # 调用API，让AI生成回复
    response = client.chat.completions.create(
        model="deepseek-chat", # 选用通用对话模型
        messages=[
            {"role": "system", "content": system_prompt}, # NPC的“内心设定”
            {"role": "user", "content": user_input}       # 玩家说的内容
        ],
        temperature=0.8, # 控制回复的随机性（0-1），数字越大越“跳脱”
        max_tokens=200    # 限制每次回复的最大长度
    )

    # 打印NPC的回复
    npc_reply = response.choices[0].message.content
    print(f"{npc_name}: {npc_reply}")
